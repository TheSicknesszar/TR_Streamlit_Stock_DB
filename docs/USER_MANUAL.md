# RefurbAdmin AI - User Manual

Complete user guide for RefurbAdmin AI.

## Getting Started

### Registration

1. Visit https://refurbadmin.co.za
2. Click "Sign Up"
3. Enter your details:
   - Email address
   - Password (min 10 characters)
   - Full name
   - Phone number (+27 format)
4. Verify your email
5. Complete your profile

### Login

1. Enter your email and password
2. Click "Login"
3. Complete 2FA if enabled

## Dashboard

The dashboard shows:
- Today's sales
- Low stock alerts
- Pending quotes
- Recent activity

## Inventory Management

### Add Product

1. Navigate to Inventory > Add Product
2. Enter product details:
   - Name
   - SKU (auto-generated)
   - Serial number
   - Category
   - Cost price (R)
   - Selling price (R)
   - Quantity
   - Condition
3. Upload product images
4. Click "Save"

### Update Stock

1. Go to Inventory
2. Find the product
3. Click "Adjust Stock"
4. Enter adjustment (+/-)
5. Select reason
6. Add notes
7. Click "Submit"

### Low Stock Alerts

Products below threshold show:
- 🔴 Critical (< 3 items)
- 🟡 Warning (< 10 items)
- 🟢 OK (≥ 10 items)

## Quotes

### Create Quote

1. Navigate to Quotes > New Quote
2. Select customer or create new
3. Add products:
   - Search products
   - Set quantities
   - Apply discounts if needed
4. Add notes
5. Set validity period
6. Click "Send Quote"

### Quote Statuses

- 📝 Draft - Not sent
- 📤 Sent - Awaiting response
- ✅ Accepted - Converted to sale
- ❌ Rejected - Customer declined
- ⏰ Expired - Past validity date

### Follow Up

1. Go to Quotes
2. Filter by "Sent"
3. Select quote
4. Click "Follow Up"
5. Choose method:
   - Email
   - WhatsApp
   - SMS
   - Phone call

## Sales

### Record Sale

1. Navigate to Sales > New Sale
2. Select customer
3. Add products
4. Choose payment method:
   - Cash
   - Card
   - EFT
   - Mzansi Instant Pay
5. Record payment
6. Generate invoice

### Invoice

Invoices include:
- Company details
- Customer details
- Itemized list
- VAT (if applicable)
- Payment terms
- QR code for payment

## Repairs

### Log Repair

1. Go to Repairs > New Repair
2. Enter customer details
3. Record device:
   - Type
   - Brand
   - Model
   - Serial number
   - Issue description
4. Set status: Received
5. Print repair tag

### Update Status

Statuses:
1. 📦 Received
2. 🔍 Diagnosing
3. ⏳ Waiting Parts
4. 🔧 In Progress
5. ✅ Completed
6. 🎉 Ready for Collection
7. 👋 Collected

### Customer Notifications

Status updates sent via:
- Email
- WhatsApp
- SMS

## Reports

### Generate Report

1. Navigate to Reports
2. Select report type:
   - Inventory
   - Sales
   - Margin Analysis
   - VAT Report
3. Set date range
4. Choose format:
   - PDF
   - Excel
   - CSV
5. Click "Generate"

### Scheduled Reports

Set up automatic reports:
1. Go to Reports > Scheduled
2. Click "New Schedule"
3. Configure:
   - Report type
   - Frequency
   - Recipients
   - Format
4. Click "Save"
