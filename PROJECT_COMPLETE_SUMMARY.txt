PROJECT OVERVIEW
================================================================================

RefurbAdmin AI is a comprehensive administrative assistant platform for an IT
repair and laptop refurbishment business in South Africa. The system automates:

✓ Inventory tracking with serial number reconciliation
✓ Dynamic pricing based on SA market conditions (PriceCheck, Takealot, Gumtree)
✓ Client communication via email and WhatsApp
✓ POPIA-compliant data handling
✓ Business analytics and reporting

CORE FEATURE: /api/v1/price-check endpoint for instant market-adjusted quotes

KNOWN LIMITATIONS & RECOMMENDATIONS

────────────────────────────────────────────────────────────────────────────────
→ Use PostgreSQL instead of SQLite (configured in docker-compose)
→ Set up Redis for caching and rate limiting
→ Configure SSL certificates (Let's Encrypt)
→ Set up proper backup rotation to cloud storage
→ Configure email SMTP credentials

### Web Scraping
────────────────────────────────────────────────────────────────────────────────

→ For production: Use rotating proxies
→ Consider official retailer APIs if available
→ Implement headless browsers for JavaScript-heavy sites
