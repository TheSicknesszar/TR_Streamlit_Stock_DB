API ENDPOINTS QUICK REFERENCE
================================================================================

Authentication:
  POST /api/v1/auth/keys           Create API key
  POST /api/v1/auth/validate       Validate API key

Inventory:
  POST   /api/v1/inventory         Create device
  GET    /api/v1/inventory         List devices
  GET    /api/v1/inventory/{id}    Get device
  PUT    /api/v1/inventory/{id}    Update device
  DELETE /api/v1/inventory/{id}    Delete device

Pricing (Core):
  POST   /api/v1/price-check       Get dynamic pricing
  GET    /api/v1/price-check       Alternative GET

Market Data:
  GET    /api/v1/market/prices     Get market prices
  POST   /api/v1/market/scrape     Trigger scraping

Monitoring:
  GET    /health                   Health check
  GET    /metrics                  Prometheus metrics
